## Installation

### Clone the project
```shell
git clone https://github.com/eipi717/BookingSystemRecommender
```

### Required Packages
1. pandas==1.5.3 
2. Requests==2.31.0 
3. scikit_learn==1.2.2

Install the required packages by running the following command:
```shell
pip install -r requirements.txt
```
